# tree tensor network strong disorder renormalization group

## tSDRG is a powerful and efficient algorithm for disorder system

### Include header file

* Operator.h
    * spin operator

* MPO.h
    * matrix product operator

* Hamiltonian.h
    * Hamiltonian model

* tSDRG_tools.h
    * tSDRG main algorithm and other tools of  tSDRG
